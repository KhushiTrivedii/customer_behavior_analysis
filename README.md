Customer Shopping Behavior Analysis

Overview:
This project analyzes customer shopping behavior using transactional data of over 3900 records to uncover insights into spending patterns, customer segmentation, product preferences, and subscription behavior.

The objective is to generate actionable insights that can support business decision making, improve customer retention, and optimize marketing strategies.

Tech Stack:
Python using pandas and numpy  
SQL using PostgreSQL  
Power BI for data visualization  

Data Cleaning and Preparation:
The dataset was processed using Python to ensure quality and consistency. Missing values in the review rating column were handled using median imputation. Column names were standardized into snake case for better readability. New features such as age groups and purchase frequency were created to enhance analysis. Redundant columns were identified and removed after consistency checks.

Data Analysis using SQL:
Structured analysis was performed in PostgreSQL to answer key business questions. These include analyzing revenue by gender, identifying high spending customers using discounts, finding top rated products, comparing subscriber and non subscriber behavior, performing customer segmentation into new, returning, and loyal groups, and evaluating revenue contribution by age group.

Dashboard Development:
An interactive dashboard was created in Power BI to present insights visually. The dashboard highlights revenue trends, customer segments, product performance, and overall shopping behavior.

Key Insights:
Loyal customers contribute significantly to overall revenue. Discount usage does not necessarily reduce customer spending. Subscription based customers show higher engagement. Certain age groups contribute more to total revenue.

Business Recommendations:
Businesses can improve subscription offerings to increase retention. Loyalty programs can help convert repeat customers into long term users. Discount strategies should be optimized to balance revenue and margins. Marketing efforts should focus on high value customer segments.

Outcome:
This project demonstrates end to end data analysis including data cleaning, feature engineering, SQL based analysis, dashboard creation, and business insight generation.

Dataset:
The dataset contains 3900 transactions with 18 features including customer demographics, purchase details, and shopping behavior attributes.
